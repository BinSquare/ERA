ERA Agent supports multiple programming languages. Each language runs in an isolated VM with its own runtime environment. This guide provides examples for each supported language.

:::tip[Using REST Clients (Postman, Insomnia, Restfox)]
When copying JSON examples to paste into REST clients, look for the **"JSON Body"** sections below each curl example. These provide single-line JSON that's ready to paste.

**Important:** Make sure newlines appear as `\n` (two characters: backslash + n) in your REST client, not as actual line breaks. If you're getting JSON parse errors about "control characters," check that `\n` isn't being converted to actual newlines.

**Example that works in REST clients:**
```json
{"language":"go","code":"package main\nimport \"fmt\"\nfunc main() { fmt.Println(\"Hello from Go!\") }"}
```
:::

## Python

Python 3.x with the standard library.

### One-Shot Execution

**curl:**
```bash
curl -X POST https://anewera.dev/api/execute \
  -H "Content-Type: application/json" \
  -d '{"language":"python","code":"print(\"Hello from Python!\")\nprint(2 + 2)"}'
```

**JSON Body:**
```json
{"language":"python","code":"print(\"Hello from Python!\")\nprint(2 + 2)"}
```

**Output:**
```
Hello from Python!
4
```

### Math and Standard Library

**curl:**
```bash
curl -X POST https://anewera.dev/api/execute \

  -d '{"language":"python","code":"import math\nprint(f\"Pi: {math.pi}\")\nprint(f\"Sqrt(16): {math.sqrt(16)}\")"}'
```

**JSON Body:**

```json
{"language":"python","code":"import math\nprint(f\"Pi: {math.pi}\")\nprint(f\"Sqrt(16): {math.sqrt(16)}\")"}
```

### Session-Based Execution

**Create session:**
```bash
curl -X POST https://anewera.dev/api/sessions \
  -H "Content-Type: application/json" \
  -d '{"language":"python","persistent":true,"session_id":"python-workspace"}'
```

**JSON Body:**
```json
{"language":"python","persistent":true,"session_id":"python-workspace"}
```

**Initialize variables:**
```bash
curl -X POST https://anewera.dev/api/sessions/python-workspace/run \
  -H "Content-Type: application/json" \
  -d '{"code":"counter = 0\nprint(f\"Counter initialized: {counter}\")"}'
```

**JSON Body:**
```json
{"code":"counter = 0\nprint(f\"Counter initialized: {counter}\")"}
```

**Variables persist:**
```bash
curl -X POST https://anewera.dev/api/sessions/python-workspace/run \
  -H "Content-Type: application/json" \
  -d '{"code":"counter += 1\nprint(f\"Counter value: {counter}\")"}'
```

**JSON Body:**
```json
{"code":"counter += 1\nprint(f\"Counter value: {counter}\")"}
```

**Clean up:**
```bash
curl -X DELETE https://anewera.dev/api/sessions/python-workspace
```

**Outputs:**
```
Counter initialized: 0
Counter value: 1
```

## Language Comparison

| Feature | Python | Node.js | TypeScript | Go | Deno |
|---------|--------|---------|------------|-----|------|
| **Runtime** | Python 3.x | Node.js | tsx | Go 1.21+ | Deno 1.40+ |
| **Typing** | Dynamic | Dynamic | Static | Static | Static/Dynamic |
| **Execution** | Interpreter | Interpreter | Transpiled | Compiled | Native TS/JS |
| **Speed** | Medium | Fast | Fast | Very Fast | Fast |
| **Use Cases** | Data analysis, scripting | APIs, async tasks | Type-safe Node.js | Systems, performance | Modern TS/JS, secure by default |

## Best Practices

### Choosing a Language

- **Python**: Data processing, scientific computing, general scripting
- **Node.js**: Async operations, JSON processing, web scraping
- **TypeScript**: Type-safe applications, large codebases
- **Go**: Performance-critical code, concurrent operations
- **Deno**: Modern TypeScript/JavaScript with native support, secure by default

### Error Handling

All languages support standard error handling:

**Python:**
```python
try:
    result = risky_operation()
except Exception as e:
    print(f"Error: {e}")
```

**Node.js:**
```javascript
try {
    const result = riskyOperation();
} catch (error) {
    console.log(`Error: ${error.message}`);
}
```

**TypeScript:**
```typescript
try {
    const result: number = riskyOperation();
} catch (error) {
    console.log(`Error: ${error}`);
}
```

**Go:**
```go
result, err := riskyOperation()
if err != nil {
    fmt.Printf("Error: %v\n", err)
    return
}
```
